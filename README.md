## Installation
Add this line to your application's Gemfile:

```ruby
gem 'gg_drive_attachment'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install gg_drive_attachment
```

install migration
```bash
rails gg_drive_attachment:install:migrations
```
